Block-Map Based Localization

ROS2 · ICP · Coarse-to-Fine Pose Estimation · Global Map Partitioning

A hierarchical robot localization system that improves scan-to-map matching efficiency by dividing a global map into overlapping block maps. The method combines coarse pose initialization using Block-Based Search (BBS) and fine alignment with Iterative Closest Point (ICP).

Gallery

Project Overview

This project implements a Block-Map Localization System to address the high computational cost of performing global scan-to-map matching in large environments. Instead of matching LiDAR scans against the full map, the environment is divided into overlapping block maps. This reduces search space, improves accuracy, and accelerates localization.

The method integrates BBS for coarse pose estimation followed by ICP for fine alignment, achieving reliable localization even with partial or noisy scan data.

Methodology

  • Partition the global occupancy grid into overlapping block maps.
  • Perform Block-Based Search (BBS) to find the most likely candidate region.
  • Extract the relevant local block map around the estimated region.
  • Run ICP alignment between the scan and block map for refined pose.
  • Evaluate results using RMSE of estimated vs ground-truth trajectory.

Algorithm Pipeline

  • Input: LiDAR scan, global occupancy map.
  • Stage 1 – Block Map Partitioning: Create block maps with overlaps of 20–40%.
  • Stage 2 – BBS: Compare scan footprint with each block using feature similarity.
  • Stage 3 – ICP: Perform fine-grained transformation refinement.
  • Stage 4 – Pose Fusion: Output final pose estimate.

Results & Observations

  • Localization accuracy improved significantly vs full-map ICP.
  • Block maps reduced search area, lowering computation time by up to 40%.
  • Overlapping blocks ensured robustness in ambiguous environments.
  • BBS provided reliable coarse initialization, reducing ICP failure cases.

Conclusion

The block-map approach demonstrates strong potential for scalable localization in large indoor environments. By combining spatial partitioning with hierarchical scan matching, the system balances efficiency and accuracy—making it suitable for SLAM backends, autonomous mobile robots, and real-time navigation frameworks.
