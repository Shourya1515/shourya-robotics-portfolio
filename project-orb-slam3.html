Visual-Inertial SLAM for Real-Time Robot Localization

ROS 2 · C++ · ORB-SLAM3 · OpenCV · Eigen · EuRoC MAV

Developed a real-time localization pipeline using ORB-SLAM3’s mono-inertial SLAM to fuse monocular vision and IMU data for accurate robot pose estimation in GPS-denied environments. The system delivers stable 30 Hz pose output with robust initialization for scale, gravity, and sensor bias estimation.

Gallery

Overview

This project implements a real-time visual-inertial SLAM system that estimates a robot’s position and orientation using only a monocular camera and an IMU. It is designed for autonomous robots operating in environments where GPS is unavailable or unreliable.

Problem

GPS performs poorly indoors and in visually challenging environments. Camera-only methods struggle with fast motion and metric scale, while IMU-only solutions drift over time. This project solves that by tightly fusing both sensing modalities.

Approach

  • Built a ROS 2 C++ node around ORB-SLAM3’s mono-inertial pipeline.
  • Used ORB feature extraction and tracking from camera images at 30 Hz.
  • Pre-integrated IMU measurements at 200 Hz for motion prediction and scale recovery.
  • Applied tightly-coupled optimization to jointly refine visual and inertial pose constraints.
  • Implemented a custom 2.5-second IMU pre-warming buffer for stable initialization of gravity, scale, and sensor biases.

Evaluation

The system was evaluated on the EuRoC MAV benchmark, a standard dataset for visual-inertial odometry research. It maintained real-time 30 Hz pose output and demonstrated robust tracking across both easy and difficult motion sequences.

Key Highlights

  • Centimeter-level position accuracy on benchmark sequences.
  • Near-perfect scale recovery, indicating successful visual-inertial initialization.
  • Stable tracking on aggressive-motion sequences where pure visual pipelines are less reliable.
  • Real-time localization suitable for downstream autonomy and navigation pipelines.

Tech Stack

  • C++ · ROS 2 · ORB-SLAM3
  • OpenCV · Eigen · Python
