Omni-Directional Robot

Kinematics · Holonomic Control · Motion Planning

An omni-wheel mobile base designed for smooth 360° motion, built around a complete kinematic model, Jacobian derivation, and inverse kinematics controller for precise holonomic motion.

Project Overview

This project focuses on the modeling and control of an omni-directional mobile robot. The base uses three or four omni wheels arranged so that the robot can translate and rotate independently, enabling extremely agile motion in tight spaces.

A full kinematic model was derived from first principles, including wheel velocity mapping, Jacobian matrices, and inverse kinematics. These equations were implemented in code to command wheel RPMs from desired body-frame velocities.

Key Features

  • Holonomic base capable of independent x, y, and yaw motion.
  • Closed-form inverse kinematics from body twist to wheel speeds.
  • Simulation tests with trajectory tracking and velocity limits.
  • Feedforward + feedback control to reduce tracking error.

Technical Stack

  • MATLAB / Python for kinematics and plotting.
  • ROS2 node for velocity commands and wheel control.
  • Rviz / simple Gazebo models for visualization.

Outcome

The omni base demonstrates smooth, intuitive motion — sliding sideways into tight gaps and rotating in place without losing position. The project serves as a foundation for future work in omni-drive platforms and indoor service robots.
