Omni Directional Mobile Robot

This project explores holonomic motion using an omni-directional drive base. The robot can translate in any direction and rotate independently, making it ideal for tight indoor spaces, warehouse logistics, and research in advanced motion planning.

Gallery

Project Overview

The robot uses omni wheels (or mecanum wheels) positioned at 90° increments to achieve full 2D mobility. By controlling the wheel velocities in a coordinated way, the platform can move laterally, diagonally, or rotate in place without needing to reorient the chassis.

The control architecture is built around a kinematic model that maps desired platform velocities to individual wheel speeds. This allows smooth transitions between different motion commands while maintaining stability and responsiveness.

Technical Breakdown

  • Omni- or mecanum-wheel drive configuration for holonomic motion
  • Kinematic model to convert platform velocity into wheel RPMs
  • Independent motor control for each wheel
  • Modular chassis design for mounting sensors and controllers
  • Designed as a testbed for advanced motion planning and control algorithms

Outcome

The omni-directional robot proved effective as a research and experimentation platform. It demonstrates strong understanding of mobile robot kinematics, drive-system design, and precise motion control for holonomic robots.
