NU Nance Report

Data Analysis · Visualization · Narrative Reporting

A structured analytical report that combines quantitative analysis with clear visuals and written narrative to support data-driven decisions.

Report Pipeline

  • Data cleaning and preprocessing.
  • Exploratory analysis to identify trends and outliers.
  • Visualization of key metrics using charts and tables.
  • Written commentary tying the data back to real decisions.

Outcome

The NU Nance report demonstrates the ability to move from raw data to a polished, stakeholder-facing document that can be used for both communication and decision support.
