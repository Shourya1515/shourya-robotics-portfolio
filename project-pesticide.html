Pesticide Spraying Robot

Agri-Robotics · Remote Operation · Safety

A manually controlled 4-wheel robot designed to help farmers spray pesticides while minimizing direct human exposure to chemicals.

Gallery

Project Overview

The robot is built on a 4-wheel platform with independent DC motors and is controlled via a remote controller. A pesticide tank and pump are mounted on the chassis, supplying fluid to the long spray pipe that extends in front of the robot.

The spraying mechanism supports both horizontal and vertical coverage using adjustable pipes and nozzles. This allows the robot to cover crop rows while the operator remains at a safer distance.

Technical Details

  • 4-wheel chassis with four motors and off-road tires.
  • On-board pesticide tank and electric pump.
  • Spray boom with multiple nozzles and adjustable height.
  • Remote-controlled motion for manual navigation in fields.

Impact

This project is an early prototype for agricultural robots that can reduce farmers’ exposure to pesticides while still using familiar, easy-to-operate controls. It also opens the door for future autonomy such as waypoint following and coverage path planning.
