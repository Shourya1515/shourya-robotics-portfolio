Pesticide Spraying Robot

Agri-Robotics · Remote Operation · Safety

A remote-controlled field robot designed to spray pesticides while reducing direct exposure for farmers. The platform uses a four-wheel drive chassis with an onboard pump and adjustable spray boom for both horizontal and vertical coverage.

Gallery

Motivation

Spraying pesticides manually exposes farmers to chemicals and results in inconsistent coverage. This project explores a low-cost robot that can be operated from a safe distance while delivering more uniform application.

Mechanical Design

  • Four-wheel chassis driven by four independent DC motors.
  • Onboard pesticide tank and pump for pressurized spraying.
  • Pipe-based boom that supports both horizontal and vertical spray patterns.

Operation

The robot is manually controlled via a remote controller, allowing the operator to stay away from the sprayed area. The boom height and orientation can be adjusted to target crops at different stages.

Impact

While this is an early prototype, it demonstrates a practical path toward more accessible agricultural robots that focus on operator safety and consistent pesticide application.
