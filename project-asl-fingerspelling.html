Real-Time ASL Fingerspelling Recognition

Python · PyTorch · OpenCV · MediaPipe · ResNet18 · MobileNetV2

Built a real-time computer vision system that recognizes 24 American Sign Language letters from webcam input at 27–30 FPS using MediaPipe hand detection and CNN-based classification. The project also investigated the domain gap between benchmark datasets and real-world webcam input.

Gallery

Overview

This project focuses on real-time sign-language perception using webcam input. The system detects a hand region using MediaPipe, preprocesses the cropped hand image, and classifies ASL fingerspelling letters using trained deep learning models.

Approach

  • Built an end-to-end webcam pipeline using OpenCV and MediaPipe hand detection.
  • Classified 24 ASL letters using CNN-based image classification.
  • Compared three architectures: custom CNN, ResNet18, and MobileNetV2.
  • Trained and evaluated models using 72,000+ benchmark training images.

Real-Time System

  • Processed live webcam frames at 27–30 FPS.
  • Used MediaPipe to localize hand landmarks and isolate the hand region.
  • Applied preprocessing to support stable classification under live camera conditions.

Results

  • Achieved 100% benchmark accuracy using ResNet18 transfer learning.
  • Compared real-time behavior across custom CNN, ResNet18, and MobileNetV2.
  • Identified a major domain shift between benchmark images and real-world webcam captures.
  • Observed a 67% accuracy drop when moving from benchmark data to real-world webcam input.

Key Learning

The most important insight from this project was that high benchmark accuracy does not automatically translate to robust real-world deployment. Lighting, camera quality, hand pose, background clutter, and user variation created a significant domain gap, reinforcing the importance of real deployment testing for perception systems.

Tech Stack
