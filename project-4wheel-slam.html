4-Wheel Autonomous Robot with SLAM & Navigation

This project demonstrates a fully autonomous 4-wheel differential drive robot built using ROS2 Foxy, SLAM Toolbox, Nav2 Navigation Stack, and a URDF-based robot model. The robot uses LiDAR for mapping, real-time localization, and path planning to autonomously navigate a multi-room indoor environment.

Gallery

Project Overview

The robot was carefully modeled in URDF with correct physical dimensions, wheel joints, and sensor placement. A simulated LiDAR scanner publishes scans used by SLAM Toolbox to generate a highly accurate indoor occupancy grid map.

Once the map is built, Nav2 handles global planning and local obstacle avoidance, allowing the robot to navigate autonomously through multiple rooms.

Technical Breakdown

  • URDF model with accurate TF frames
  • SLAM Toolbox for mapping
  • Nav2 for global + local planning
  • Costmaps for safe obstacle avoidance
  • Differential-drive odometry
  • RViz visualization of map, laserscan, and path

Outcome

This project demonstrates my ability to integrate ROS2 navigation, robot modeling, SLAM algorithms, and real-time visualization for autonomous robotics applications.
