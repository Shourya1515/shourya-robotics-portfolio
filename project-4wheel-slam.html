4-Wheel Autonomous Robot with SLAM & Navigation

ROS2 Foxy · SLAM Toolbox · Nav2 Navigation Stack · Gazebo

A fully autonomous 4-wheel differential drive robot that performs LiDAR-based mapping, localization, and path planning across a multi-room indoor environment using a URDF robot model, SLAM Toolbox, and the Nav2 navigation stack.

Gallery

Project Overview

The 4-wheel robot is modeled in URDF with accurate wheelbase, wheel radius, and LiDAR placement. Using Gazebo for simulation and RViz for visualization, the robot publishes odometry from a differential-drive model and laser scans from a simulated 2D LiDAR.

SLAM Toolbox consumes the laser scans and odometry to build a high-resolution 2D occupancy grid map spanning multiple rooms. Once mapping is complete, the Nav2 stack takes over for autonomous navigation to user-defined goal points.

Key Features

  • URDF model with base_link, wheel joints, LiDAR frame, and collision meshes.
  • SLAM Toolbox with loop-closure enabled for drift correction.
  • Nav2 global and local planners for path generation.
  • Global and local costmaps with inflation layers around obstacles.
  • TF tree linking map → odom → base_link → laser frames.

Technical Stack

  • ROS2 Jazzy / Nav2 / SLAM Toolbox
  • Gazebo simulation + RViz visualization
  • Differential-drive kinematics for odometry
  • AMCL / SLAM pose estimation for localization
  • Behavior-tree based navigation (bt_navigator)

Outcome

The robot successfully explores and maps a complex indoor layout, then uses the saved map to repeatedly navigate to 2D Nav Goals with stable localization and smooth collision-free motion. This project showcases end-to-end capability in robot modeling, SLAM, and autonomous navigation.
