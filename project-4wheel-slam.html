4-Wheel Autonomous Robot with SLAM & Navigation

A fully autonomous mobile robot built using ROS2 Foxy, SLAM Toolbox, Nav2 Navigation Stack, and a complete URDF robot model. The robot performs LiDAR-based mapping, localization, and autonomous path planning across a multi-room indoor environment.

Gallery

Project Overview

The robot’s URDF model includes accurate joint frames, wheel kinematics, and LiDAR placement. Using SLAM Toolbox, the robot builds a multi-room occupancy grid map with loop closure. The Nav2 stack then computes global paths and performs safe real-time obstacle avoidance using layered costmaps, allowing the robot to autonomously move between goal points.

Technical Breakdown

  • URDF model with complete TF tree (base_link, wheels, LiDAR, odom, map).
  • SLAM Toolbox for mapping and pose estimation with loop closure.
  • Nav2 global (NavFn/Smac) and local planners with behavior trees.
  • Inflation-based costmaps for obstacle avoidance and safe path generation.
  • Differential-drive odometry computed from wheel velocities.
  • Full RViz integration for map, laser scans, costmaps, and trajectories.

Outcome

The system demonstrates a complete robotics pipeline from modeling to autonomy, and highlights my experience with ROS2 navigation, SLAM, TF coordination, and system integration.
